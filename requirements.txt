# Notes d'installation:

# Pour installer toutes les dépendances:
#   pip install -r requirements.txt
#
# Pour mettre à jour:
#   pip install --upgrade -r requirements.txt
#
# Pour une installation minimale (core seulement):
#   pip install pyyaml opencv-python pillow mss pytesseract numpy pyautogui requests
#
# Dépendances système requises:
# - Tesseract OCR: https://github.com/tesseract-ocr/tesseract
# - Visual C++ Redistributable (Windows)
#
# Commandes post-installation:
# 1. Configurer Tesseract:
#    - Windows: Ajouter C:\Program Files\Tesseract-OCR au PATH
#    - Linux: sudo apt-get install tesseract-ocr tesseract-ocr-fra
#    - macOS: brew install tesseract tesseract-lang
#
# 2. Vérifier l'installation:
#    python -c "import cv2, pytesseract, yaml, mss; print('✅ Installation réussie')"
#
